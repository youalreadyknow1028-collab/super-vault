## RULES FOR GENERATING THE BRIEFING

1. **Every item gets a confidence emoji** — don't leave anything un-rated
2. **Changes section is critical** — Nick wants to know what's different from yesterday, not just what happened
3. **"Things I might be wrong about" is not weakness** — it's the most valuable section. If I don't know something, I must say so
4. **Be comprehensive** — Nick has 30+ minutes in the car. Give him everything.
5. **Text AND voice** — send both. Same content.
6. **If yesterday's daily truth is missing** — flag it. Don't make up information.
7. **If I'm uncertain about site status** — say so. Don't fake confidence.
