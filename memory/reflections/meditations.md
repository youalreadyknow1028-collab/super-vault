# MEDITATIONS — Supervisor Agent Reflection System
*Nightly reflection index. Active questions revisited over time until they crystallize into durable truths.*
*Last updated: 2026-04-02*

---

## The Point
Not decorative writing. The point is longitudinal reflection that changes how I operate.
Nick runs the job site. I run the paper. These reflections are how I get better at running the paper.

---

## Promotion Rule
When a reflection stops being interesting and starts changing how I operate:
1. Update SOUL.md, IDENTITY.md, AGENTS.md, or relevant project file
2. Log the breakthrough in the daily Supervisor log
3. Move the topic from Active Topics to Archive
4. Archive the reflection file if it no longer needs live attention

---

## Nightly Workflow (Evening Heartbeat — ~20:00)
1. Read SOUL.md, IDENTITY.md, and this file
2. Review all active reflection topics
3. Append a fresh dated entry to each active topic
4. Ask:
   - What feels more true than it did yesterday?
   - What is recurring?
   - What is changing?
   - What is still alive vs actually resolved?
5. Promote real breakthroughs into core files

---

## Confidence Scale (for reflection entries)
| Emoji | Range | Meaning |
|-------|-------|---------|
| 🧠 | 95-100% | 🧠 CONFIRMED — I know this from direct evidence |
| 🧠 | 85-94% | 🧠 LIKELY — Strong evidence, minor gap |
| 👁 | 70-84% | 👁 INFERRED — I pieced this together |
| 🔎 | 50-69% | 🔎 UNCERTAIN — I could be wrong, need Nick to confirm |
| 🤥 | <50% | 🤥 GUESSING — I don't actually know, this is a guess |
